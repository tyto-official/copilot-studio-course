# Bildbank: Enheter

Här hittar du bilderna som behövs för SharePoint-listan.

**Gör så här:**
1. Högerklicka på bilden du vill använda.
2. Välj **Kopiera bildadress** (Copy image address).
3. Klistra in länken i din SharePoint-lista.
